Sobre o Projeto

Aluno: ALISSON VITOR LOHN VODZICKI

Tema escolhido: Catálogo de Disciplinas Acadêmicas

Objetivo do sistema: permitir que o usuário visualize disciplinas acadêmicas e adicione matérias à grade de interesse.

Tecnologias usadas: Angular, TypeScript, HTML e CSS.

Requisitos demonstrados

  • Rotas: Home, Listagem, Favoritos e Sobre.
  • Menu com navegação usando routerLink.
  • Componente filho reutilizável para os cards.
  • Uso de @Input() para enviar dados do pai para o filho.
  • Uso de @Output() para enviar eventos do filho para o pai.
  • Lista local em memória usando array de objetos.
  • Adição, visualização e remoção de disciplinas da grade de interesse.
